# FFB Business Focus Quiz - v3c (Soft White-Space)

Tracy Harris Co. Freedom Filled(R) Business Focus assessment. Soft white-space treatment: 80% oatmeal + white canvas, everything breathes. Large rounded corners, sage hairlines only, per-pillar color dots (not fills). Results page reads like a calm personal letter. Meditation / reset tool energy.

## Live

https://purepurposekh.github.io/ffb-quiz-v3c/

## Stack

Vanilla HTML + CSS + JS. Brand tokens and fonts loaded from the shared brand kit via jsDelivr.

## Files

- `index.html` - airy landing (centred, small round portrait, per-pillar dots, pill CTAs)
- `quiz.html` - 20 question flow, rounded cream answer cards with sage focus ring
- `results.html` - "Hey lovely" letter layout with per-pillar dots, focus note, three wins, Tracy sign-off
- `styles.css` - soft white-space treatment
- `assets/img/tracy-hero.jpg` - portrait photo (small circular)
